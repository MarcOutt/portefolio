    Le système de surveillance des prix Books Online vise à extraire des informations sur les livres à partir du site Books to Scrape, à les stocker dans un format structuré et à télécharger les images correspondantes.

    Extraction des informations d'un livre

    • Le système doit extraire les informations d'une page Produit du site Books to Scrape (URL de la page produit, Code produit universel (UPC), Titre, Prix TTC, Disponibilité, Description du produit, Catégorie, Note de revue, URL de l'image).
    • Les données extraites doivent être enregistrées dans un fichier CSV avec des en-têtes de colonnes correspondants.

    Extraction des informations d'une catégorie

    • Le système doit extraire les informations produit de tous les livres appartenant à une catégorie spécifiée.
    • Les données de tous les livres de la catégorie doivent être enregistrées dans un fichier CSV unique.
    • Gérer automatiquement la pagination des pages de catégorie, le cas échéant.

    Extraction des informations de toutes les catégories

    • Le système doit extraire les informations produit de tous les livres de toutes les catégories disponibles sur Books to Scrape.
    • Les données de chaque catégorie doivent être enregistrées dans un fichier CSV distinct.

    Téléchargement des images des pages Produit

    • Le système doit télécharger et enregistrer localement les images de chaque page Produit consultée.
    • Les images doivent être organisées par catégorie dans des dossiers distincts.
